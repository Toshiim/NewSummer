## Заметки разработки
Реализованно уже:
- Доменная модель
- EF конфигурация
- 3 эндпоинта для управления сервисом(Get: /articles, /sources. Post /sources)
- RSS парсинг 
- Кравлинг статей(черезе [SmartReader](https://github.com/Strumenta/SmartReader))
- Суммаризация без категоризации

Парсинг RSS и кравлинг статей работает с любыми источниками(тестилось на BBC, Habr, DTF).
